\begin{abstract}
    Linear response theory (LRT) provides a powerful framework for predicting the response of a system to small perturbations from information contained in its unperturbed state. Here, we develop a steady-state and time-dependent linear response framework for noisy network dynamics near stable fixed points. The network is modeled as coupled stochastic variables with intrinsic node dynamics, weighted and directed interactions, and Gaussian white noise. By linearizing the dynamics around the noise-free fixed point, we formulate the theory for small perturbations in network parameters. The framework is applied to both equilibrium and nonequilibrium steady states, where detailed balance is broken by asymmetric couplings and nonuniform noise strengths. We derive explicit response functions and verify them by measuring the change in the equal-time covariance $\langle \delta x_i \delta x_j \rangle$ before and after the perturbation in Langevin simulations. We also extend the formalism to time-dependent parameter changes. Our results provide a unified description of steady-state and time-dependent response in noisy network dynamics near stable fixed points, and a predictive framework for perturbation response in both equilibrium and nonequilibrium settings.
\end{abstract}
