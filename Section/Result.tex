\section{Numerical Validation of the Steady-State Response Theory}
\subsection{Simulation setup and observables}
\quad We perform Langevin simulations on Erd\H{o}s-R\'enyi networks to validate the steady-state response formulas. 
For each realization, the adjacency matrix is generated independently with connection probability $p$, and the nonzero couplings and node parameters are then assigned from prescribed distributions specified below for each figure. 
The dynamics are integrated with time step $\Delta t=0.0005$ up to time $T=10000$. We simulate both the unperturbed system and the corresponding perturbed system as two separate Langevin trajectories. The unperturbed trajectory is used to estimate the reference steady-state covariance matrix $\mathbf{K}_0$, while the perturbed trajectory is evolved to the new steady state and used to estimate the perturbed covariance matrix $\mathbf{K}_0^{\rm pert}$. The measured response is then defined as $\Delta \mathbf{K}_0 = \mathbf{K}_0^{\rm pert} - \mathbf{K}_0$.
Starting from the same unperturbed network realization, a node perturbation is implemented by replacing $r_\alpha$ with $r_\alpha+\delta r_\alpha$, while a noise perturbation is $\sigma_{\alpha\alpha}$ with $\sigma_{\alpha\alpha}+\delta \sigma_{\alpha\alpha}$. An edge perturbation is implemented by replacing $W_{\alpha\beta}$ with $W_{\alpha\beta}+\delta W_{\alpha\beta}$. For undirected networks, the reciprocal edge is changed by the same amount, $W_{\beta\alpha}\to W_{\beta\alpha}+\delta W_{\alpha\beta}$, so that the symmetric topology is preserved after perturbation. Each panel is obtained from a single network realization. Each scatter point corresponds to one covariance-matrix element $(\Delta \mathbf{K}_0)_{ij}$ from that realization, and different marker shapes denote different perturbation amplitudes when more than one amplitude is shown.
For the steady-state validation, we use Erd\H{o}s-R\'enyi networks with $N=50$ or $100$ and connection probability $p \simeq 0.1\text{--}0.2$. The intrinsic dynamics are chosen so that the deterministic system has a stable fixed point, and the coupling is taken to be synchronizing. We consider two representative classes of intrinsic dynamics: the linear restoring force $f_i(x_i)=-r_i(x_i-1)$ and the Logistic force $f_i=r_i x_i(1-x_i)$, both with coupling function $h(x_i,x_j)=x_j-x_i$. Around the stable fixed point $X_i=1$, both choices lead to the same Jacobian matrix,
\begin{equation}
    Q_{ii} = -r_i - \sum_{j\neq i} W_{ij},\; Q_{ij} = W_{ij},
\end{equation}
where the noise-free fixed point of each node is $1$. We therefore use the linear system as a benchmark for validating the closed-form equilibrium response formulas, and then use the Logistic system to verify that the same theory applies to a nonlinear dynamics through its linearization near the stable fixed point. Detailed balance is satisfied for undirected networks with uniform noise strengths, whereas it is broken either by asymmetric couplings or by nonuniform diagonal noise strengths. Throughout this section, the node parameters and couplings are chosen so that the linearized dynamics remains stable. For the validations in the following figures, the nonzero couplings are sampled from a Gaussian distribution $W_{ij}\sim\mathcal{N}(5,0.5)$. In the equilibrium benchmarks with uniform noise, the node parameters are sampled independently from the interval $[2.5,7.5]$ and the diagonal noise strengths are fixed at $\sigma_{ii}=0.01$. In the NESS benchmarks, the same sampling protocol is used for the node parameters and nonzero couplings; detailed balance is then broken either by asymmetric couplings with uniform noise or by using a nonuniform positive diagonal noise matrix.\par

\subsection{Equilibrium response: undirected networks with uniform noise}
\quad We first consider undirected networks with uniform noise strengths, for which the steady state satisfies detailed balance. This part has two purposes. We first use the linear restoring-force system as a benchmark to test the closed-form equilibrium formulas (\ref{chiKodr_eq}) and (\ref{chiKodW_eq}) for perturbations of a single node parameter or a single edge weight. We then turn to the Logistic system to verify that the same response theory remains valid for a nonlinear dynamics once it is linearized near the stable fixed point. Figure \ref{chieqmlinear}a and b compare the measured change in $\mathbf{K}_0$ with the theoretical prediction for a randomly chosen node or edge in the linear benchmark. The noise strengths are identical among the nodes, with $\sigma_{ii}=0.01$. In both panels, the data points collapse onto the diagonal, showing agreement with the equilibrium response formulas for the representative realization considered here. In Fig. \ref{chieqmlinear}a, the perturbed node $\alpha$ is tested with three values of $\delta r_\alpha = 0.1, 0.5, 2$. Increasing $r_\alpha$ strengthens the local restoring force and therefore reduces the fluctuation level, with the largest change occurring at $(\mathbf{K}_0)_{\alpha\alpha}$.

\subsection{Nonequilibrium steady-state response}
\quad In our framework, a NESS arises when detailed balance is broken, which can occur either through asymmetric interactions or heterogeneous noise distributions. In contrast to equilibrium systems, where the steady state is characterized by vanishing probability currents, NESS exhibit persistent irreversible fluxes. The key question is whether the steady-state linear response framework remains predictive under such conditions. Before comparing with simulations, it is important to show that the equilibrium-like simplification is no longer sufficient once detailed balance is broken. To this end, we compare the equilibrium-like formulas (\ref{chiKodr_eq}) and (\ref{chiKodW_eq}) with the exact NESS response obtained from the Lyapunov equation. If the equilibrium-like approximation remained valid, the data would collapse onto the diagonal in a plot of $\chi^{eq}$ versus $\chi^{ss}$. Figure \ref{chieqcompare} shows that this is not the case: systematic deviations from the diagonal appear both when detailed balance is broken by heterogeneous noise strengths [panels (a) and (b)] and when it is broken by asymmetric couplings [panels (c) and (d)]. The discrepancy demonstrates that the full Lyapunov response equation is essential in NESS and is not merely a small correction to the equilibrium form.\par
Similar to the equilibrium case, we compare simulation results with the theoretical response functions under two NESS conditions: a directed graph with a uniform diagonal noise matrix in Fig. \ref{chiKodir}, and an undirected graph with a nonuniform diagonal noise matrix in Fig. \ref{chiKonon}. In both cases, the perturbed node and the perturbed edge are randomly chosen, and the perturbation amplitudes are $\delta r_\alpha = 0.5, 2$ and $\delta W_{\alpha\beta} = 0.5, 2$. The data collapse onto the diagonal, showing that the equal-time covariance response is accurately captured for the representative realizations considered here even when detailed balance is broken. In the directed graph, the edge perturbation acts on a single directed edge. In the undirected graph, both $W_{\alpha\beta}$ and $W_{\beta\alpha}$ are perturbed by the same amount to preserve the symmetric topology. In Fig. \ref{chiKodir}b, the perturbed edge points from node $\beta$ to node $\alpha$, so the strongest response appears at $(\mathbf{K}_0)_{\alpha\alpha}$. In Fig. \ref{chiKonon}b, both nodes $\alpha$ and $\beta$ are directly affected, and the strongest negative responses occur in $(\mathbf{K}_0)_{\alpha\alpha}$ and $(\mathbf{K}_0)_{\beta\beta}$.

\subsection{Multiple perturbations}
\quad After validating the linear-response theory for single-parameter perturbations, we next examine whether the response functions can be superposed to predict the effect of multiple simultaneous perturbations. For multiple perturbations, the perturbations are applied simultaneously to all selected nodes or edges. Edge removal is realized by setting the chosen edge weights to zero, corresponding to $\delta W_{\alpha\beta}=-W_{\alpha\beta}$, and node removal is realized by isolating the chosen node through setting all incident edge weights to zero. We define the covariance change as
\begin{equation}
    \Delta (\mathbf{K}_0)_{ij} =
    (\mathbf{K}_0^{\rm after})_{ij}-(\mathbf{K}_0^{\rm before})_{ij}.
\end{equation}
For a set of perturbed nodes
\(V^*=\{\alpha_1,\alpha_2,\ldots,\alpha_m\}\), the first-order prediction is
\begin{equation}
    \Delta (\mathbf{K}_0)_{ij} = \sum_{\alpha\in V^*}
    \chi^{(r)}_{ij\alpha}\delta r_\alpha .
\end{equation}
Similarly, for a set of perturbed edges \(E^*=\{(\alpha_1,\beta_1),(\alpha_2,\beta_2),\ldots, (\alpha_m,\beta_m)\}\), we use
\begin{equation}
    \Delta (\mathbf{K}_0)_{ij} = \sum_{(\alpha,\beta)\in E^*} \chi^{(W)}_{ij(\alpha\beta)}\delta W_{\alpha\beta}.
\end{equation}
The perturbation amplitudes \(\delta r_\alpha\) and \(\delta W_{\alpha\beta}\) are allowed to be different for different perturbed nodes or edges. To test this superposition property, we perform Langevin simulations on Erd\H{o}s-R\'enyi networks with nonuniform noise sources. In each realization, multiple nodes or edges are randomly selected and perturbed simultaneously. The covariance change measured from the simulations is then compared with the first-order prediction obtained by summing the corresponding single-parameter response functions. Agreement with the diagonal line in the scatter plots indicates that the covariance response remains additive within the linear regime for the realization tested.\par
In Fig. \ref{MultipledK}, twenty randomly selected nodes are perturbed with the same amplitude $\delta r_\alpha=0.5$, where the set of perturbed nodes is denoted by $V^*=\{ \alpha_1, \alpha_2, ..., \alpha_{10} \}$. For visual clarity, the covariance elements are separated into diagonal and off-diagonal components, and entries involving perturbed nodes are distinguished from those that do not. The data points collapse onto the diagonal line, indicating quantitative agreement between the linear-response prediction and the numerical simulation for multiple node perturbations in the realization considered here. Forty randomly selected undirected edges are perturbed with the same amplitude $\delta W_{\alpha\beta}=0.5$. The perturbed edges are represented as the set $E^* = \{ (\alpha_k,\beta_k),\;k=1,2,...,20 \}$. The data again align along the diagonal, confirming that the first-order response remains additive for multiple edge perturbations within the linear regime for the realization shown.\par
To further quantify the range of validity of the superposition approximation, Fig. \ref{NRMSE_Error_Perturb} reports the normalized root-mean-square error (NRMSE) between the predicted covariance response and the simulation result as a function of the number of simultaneously perturbed nodes or edges. We consider an undirected Erd\H{o}s-R\'enyi network with \(N=100\), \(p=0.1\), and nonuniform noise strengths, and randomly perturb \(10\text{--}50\) nodes or \(20\text{--}100\) undirected edges. The perturbation amplitudes are fixed at \(\delta r_{\alpha}=1\) for node perturbations and \(\delta W_{\alpha\beta}=1\) for edge perturbations. The NRMSE remains small over a broad range of perturbation sizes, indicating that the linear-response prediction continues to provide a good approximation even when as many as \(N/2\) nodes are perturbed or when the number of perturbed edges reaches approximately \(20\%\) of the total number of edges.

\subsection{Stress test for removal of nodes and edges}
We next consider removal events as large perturbation stress tests of the linear-response framework. Unlike the multiple-perturbation tests above, these removal protocols are not infinitesimal perturbations, and therefore they are not expected to provide a quantitatively exact validation of the first-order theory. Instead, they are used to identify the regime in which the linear response remains informative and the regime in which higher-order effects become important. Edge removal is implemented by setting \(W_{\alpha\beta}\to 0\), corresponding to \(\delta W_{\alpha\beta}=-W_{\alpha\beta}\). Node removal is implemented by isolating the selected node through the removal of all incoming and outgoing edges connected to it. Figure \ref{MultipledKdW_remove} examines the removal of 40 directed edges in a directed Erd\H{o}s-R\'enyi network with nonuniform noise. The prediction is obtained by summing the single-edge response functions over all removed edges. The approximate agreement with the diagonal shows that, for this level of edge removal, the first-order response still captures the dominant covariance change. Figure \ref{MultipledKdW_noderemove} considers the removal of a single node in an undirected network with nonuniform noise. Because this operation simultaneously deletes all edges connected to the node, it represents a large perturbation. The visible deviations from the diagonal are therefore expected and indicate the onset of higher-order response terms beyond the first-order theory.

Figure \ref{NRMSE_Error_Remove} provides a complementary error-based assessment for removal-type perturbations. In Fig. \ref{NRMSE_Error_Remove}a, we randomly remove \(10\text{--}50\) undirected edges from an undirected Erd\H{o}s-R\'enyi network with \(N=100\), \(p=0.1\), and nonuniform noise strengths, and plot the NRMSE between the first-order prediction and the simulation result as a function of the number of removed edges. In Fig. \ref{NRMSE_Error_Remove}b, we examine node removal in a controlled manner by scaling all edges connected to a selected node according to \(\delta W_{\alpha\beta}=-\lambda W_{\alpha\beta}\), where \(\lambda=1\) corresponds to complete removal. The results show that the first-order response remains informative for edge removal as long as the number of removed edges is only a small fraction of the total, approximately \(2\%\) in the present example. For node weakening or removal the error grows rapidly once \(\lambda>0.2\), indicating that the resulting perturbation has already left the linear-response regime. This behavior highlights the intrinsic limitation of the first-order framework for large structural perturbations and motivates the development of a nonlinear response theory.
