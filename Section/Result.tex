\section{Numerical Validation of the Steady-State Response Theory}
\subsection{Simulation setup and observables}
\quad We validate the steady-state response formulas by Langevin simulations on Erd\H{o}s-R\'enyi networks with $N=50,100$ and connection probability $p\simeq 0.1$--$0.2$. The dynamics are integrated by the Euler--Maruyama method with time step $\Delta t=0.0005$ up to $T=10000$. We use the two benchmark models introduced in Sec.~IV: the linear restoring-force system, which provides an exact Ornstein-Uhlenbeck benchmark, and the logistic system, which tests how well the fixed-point approximation performs for nonlinear dynamics. To quantify the network response, we simulate both the unperturbed and perturbed systems. The unperturbed trajectory yields the reference covariance matrix $\mathbf{K}_0^{\rm before}$, while the perturbed trajectory yields $\mathbf{K}_0^{\rm after}$. The measured response is
\begin{equation}
    \Delta \mathbf{K}_0 = \mathbf{K}_0^{\rm after}-\mathbf{K}_0^{\rm before} .
\end{equation}
Node, edge, and noise perturbations are implemented as
\begin{equation}
    r_\alpha\to r_\alpha+\delta r_\alpha,\qquad
    W_{\alpha\beta}\to W_{\alpha\beta}+\delta W_{\alpha\beta},
    \qquad
    \sigma_{\alpha\alpha}\to \sigma_{\alpha\alpha}+\delta\sigma_{\alpha\alpha}.
\end{equation}
For undirected networks, the reciprocal edge is perturbed by the same amount,
$W_{\beta\alpha}\to W_{\beta\alpha}+\delta W_{\alpha\beta}$, to preserve symmetry. Each panel shows one representative network realization; each scatter point corresponds to one covariance element $(\Delta\mathbf{K}_0)_{ij}$, and different markers denote different perturbation amplitudes when applicable. For each perturbation test, the theoretical prediction (Eq. \ref{chiKodr_ss}, \ref{chiKodW_ss}, \ref{partialKo_sigma}) is constructed exclusively from the unperturbed reference state measured in simulations, using the benchmark assumptions summarized in Sec.~IV. Unless stated otherwise, nonzero couplings are sampled from $W_{ij}\sim\mathcal{N}(5,0.5)$, and node parameters are sampled independently from $[2.5,7.5]$. Throughout this section, parameters are chosen so that the linearized dynamics remains stable.
\subsection{Equilibrium response: undirected networks with uniform noise}
\quad We first consider undirected networks with uniform diagonal noise strengths, for which the steady state satisfies detailed balance. In this equilibrium setting, it is useful to distinguish perturbations by which part of the linearized dynamics they modify. A node perturbation changes the diagonal part of $\mathbf{Q}$ through the local restoring rate, whereas an edge perturbation changes the off-diagonal coupling structure. A perturbation of $\sigma_{\alpha\alpha}$ is qualitatively different: it does not modify $\mathbf{Q}$, but directly changes the local fluctuation input. If the unperturbed system satisfies detailed balance, the corresponding linear susceptibility can still be evaluated from the equilibrium reference state. However, changing $\sigma_{\alpha\alpha}$ generally breaks the uniform-noise condition, so the perturbed system need not remain in the equilibrium class. For this reason, we focus here on node and edge perturbations, for which the perturbed undirected network with uniform noise remains within the detailed-balance class.\par
Fig. \ref{chieqmlinear}a and b compare the measured change in $\Delta \mathbf{K}_0 / \delta p_\alpha$ with the theoretical prediction for a randomly chosen node or edge in the linear benchmark. In both panels, the data points collapse onto the diagonal, showing agreement with the equilibrium response formulas for the representative realization considered here. In Fig. \ref{chieqmlinear}a, the perturbed node $\alpha$ is tested with three values, $\delta r_\alpha = 0.1, 0.5, 2$. For the case $\delta r_\alpha=2$, the NRMSE of $\chi_{ij\alpha}$ is approximately $5.4\%$. Increasing $r_\alpha$ strengthens the local restoring force and therefore reduces the fluctuation level, with the largest change occurring at $(\mathbf{K}_0)_{\alpha\alpha}$. In Fig. \ref{chieqmlinear}b, the perturbed edge $(\alpha,\beta)$ is chosen at random, and three positive perturbation values are tested: $\delta W_{\alpha\beta} = 0.2, 1, 5$. For $\delta W_{\alpha\beta}=1$ and $5$, the corresponding NRMSEs of $\chi_{ij\alpha\beta}$ are approximately $5.01\%$ and $19.9\%$, respectively, with $\delta W_{\alpha\beta}=5$ already lying beyond the linear-response regime. For an undirected network, $W_{\beta\alpha}$ is changed by the same amount to preserve the symmetric topology. Increasing the coupling between nodes $\alpha$ and $\beta$ enhances their correlation, so the largest positive response appears in $(\mathbf{K}_0)_{\alpha\beta}$, while the auto-correlations $(\mathbf{K}_0)_{\alpha\alpha}$ and $(\mathbf{K}_0)_{\beta\beta}$ decrease. Fig. \ref{chieqmlogistic} shows that the same equilibrium response formulas continue to agree with the simulations for the nonlinear benchmark as well.

\subsection{Nonequilibrium steady-state response}
\subsubsection*{Breakdown of equilibrium-like response in NESS}
\quad In our framework, a NESS arises when detailed balance is broken, either through asymmetric interactions or through heterogeneous noise distributions. In contrast to equilibrium systems, where the steady state is characterized by vanishing probability currents, NESS exhibit persistent irreversible fluxes. The key question is whether the steady-state linear-response framework remains predictive under such conditions. Before comparing with simulations, we first show that the equilibrium-like simplification is no longer sufficient once detailed balance is broken. To this end, we compare the equilibrium-like formulas (\ref{chiKodr_eq}) and (\ref{chiKodW_eq}) with the exact NESS response obtained from the Lyapunov equation. If the equilibrium-like approximation remained valid, the data would collapse onto the diagonal in a plot of $\chi^{eq}$ versus $\chi^{ss}$. Fig. \ref{chieqcompare} shows that this is not the case: systematic deviations from the diagonal appear both when detailed balance is broken by heterogeneous noise strengths [panels (a) and (b)] and when it is broken by asymmetric couplings [panels (c) and (d)]. This discrepancy shows that the full Lyapunov response equation is essential in NESS, rather than merely a small correction to the equilibrium form.\par

\subsubsection*{Validation of the full Lyapunov response in NESS}
We next compare how different perturbation channels act on two representative nonequilibrium backgrounds. The first is a directed graph with a uniform diagonal noise matrix, where detailed balance is broken by asymmetric couplings. The second is an undirected graph with a nonuniform diagonal noise matrix, where detailed balance is broken by heterogeneous noise strengths. In both cases, the perturbed node and the perturbed edge are chosen at random, and the perturbation amplitudes are $\delta r_\alpha = 0.5, 2$ and $\delta W_{\alpha\beta} = 0.5, 2$. The NRMSEs for the larger perturbations, $\delta r_\alpha = 2$ and $\delta W_{\alpha\beta} = 2$, are $4.2\%$ and $6.7\%$, respectively. The data collapse onto the diagonal, showing that the equal-time covariance response is accurately captured for the representative realizations considered here even when detailed balance is broken. In the directed graph, the edge perturbation acts on a single directed edge. In the undirected graph, both $W_{\alpha\beta}$ and $W_{\beta\alpha}$ are perturbed by the same amount to preserve the symmetric topology. In Fig. \ref{chiKodir}b, the perturbed edge points from node $\beta$ to node $\alpha$, so the strongest response appears at $(\mathbf{K}_0)_{\alpha\alpha}$. In Fig. \ref{chiKonon}b, both nodes $\alpha$ and $\beta$ are directly affected, and the strongest negative responses occur in $(\mathbf{K}_0)_{\alpha\alpha}$ and $(\mathbf{K}_0)_{\beta\beta}$. Fig. \ref{chiKonon}c then highlights the distinctive role of the noise channel, with $\delta \sigma_{\alpha\alpha} = 0.002, 0.01$; for the larger perturbation, $\delta \sigma_{\alpha\alpha} = 0.01$, the NRMSE is $2.5\%$. Agreement with the diagonal in this panel therefore shows that the NESS response theory captures not only structural perturbations of the deterministic dynamics but also direct perturbations of the noise source itself.

\subsection{Multiple perturbations}
\quad After validating the response to single-parameter perturbations, we next examine the practical range over which first-order responses can be superposed for multiple simultaneous perturbations. For a set of small parameter changes \(\{\delta p_a\}\), the covariance response admits a first-order additive expansion. In the simulations, the perturbations are applied simultaneously to all selected nodes or edges. We define the covariance change as
\begin{equation}
    \Delta (\mathbf{K}_0)_{ij} =
    (\mathbf{K}_0^{\rm after})_{ij}-(\mathbf{K}_0^{\rm before})_{ij}.
\end{equation}
For a set of perturbed nodes
\(V^*=\{\alpha_1,\alpha_2,\ldots,\alpha_m\}\), the first-order prediction is
\begin{equation}
    \Delta (\mathbf{K}_0)_{ij} = \sum_{\alpha\in V^*}
    \chi^{(r)}_{ij\alpha}\delta r_\alpha .
\end{equation}
Similarly, for a set of perturbed edges \(E^*=\{(\alpha_1,\beta_1),(\alpha_2,\beta_2),\ldots, (\alpha_m,\beta_m)\}\), we use
\begin{equation}
    \Delta (\mathbf{K}_0)_{ij} = \sum_{(\alpha,\beta)\in E^*} \chi^{(W)}_{ij(\alpha\beta)}\delta W_{\alpha\beta}.
\end{equation}
For undirected networks, perturbing an undirected edge $\{\alpha, \beta\}$ changes both $W_{\alpha\beta}$ and $W_{\beta\alpha}$, so $\chi^{(W,\text{und})}_{ij\{\alpha\beta\}} = \chi^{(W)}_{ij(\alpha\beta)}+\chi^{(W)}_{ij(\beta\alpha)}$. The perturbation amplitudes \(\delta r_\alpha\) and \(\delta W_{\alpha\beta}\) are allowed to vary across the perturbed nodes or edges. To test this superposition property, we perform Langevin simulations on Erd\H{o}s-R\'enyi networks with nonuniform noise sources. In each realization, multiple nodes or edges are randomly selected and perturbed simultaneously. The covariance change measured from the simulations is then compared with the first-order prediction obtained by summing the corresponding single-parameter response functions. Agreement with the diagonal line in the scatter plots indicates that the covariance response remains additive within the linear regime for the realization tested.\par
In Fig. \ref{MultipledK}, twenty randomly selected nodes are perturbed with the same amplitude $\delta r_\alpha=0.5$, where the set of perturbed nodes is denoted by $V^*=\{ \alpha_1, \alpha_2, ..., \alpha_{20} \}$. For visual clarity, the covariance elements are separated into diagonal and off-diagonal components, and entries involving perturbed nodes are distinguished from those that do not. The data points collapse onto the diagonal line, indicating quantitative agreement between the linear-response prediction and the numerical simulation for multiple node perturbations in the realization considered here. Forty randomly selected undirected edges are likewise perturbed with the same amplitude $\delta W_{\alpha\beta}=0.5$. The perturbed edges are represented as the set $E^* = \{ (\alpha_k,\beta_k),\;k=1,2,...,40 \}$. The data again align along the diagonal, confirming that the first-order response remains additive for multiple edge perturbations within the linear regime for the realization shown.\par
To further quantify the range of validity of the superposition approximation, Fig. \ref{NRMSE_Error_Perturb} reports the normalized root-mean-square error (NRMSE) between the predicted covariance response and the simulation result as a function of the number of simultaneously perturbed nodes or edges. We consider an undirected Erd\H{o}s-R\'enyi network with \(N=100\), \(p=0.1\), and nonuniform noise strengths, and randomly perturb \(10\text{--}50\) nodes or \(20\text{--}100\) undirected edges. The perturbation amplitudes are fixed at \(\delta r_{\alpha}=1\) for node perturbations and \(\delta W_{\alpha\beta}=1\) for edge perturbations. The NRMSE remains small over a broad range of perturbation sizes, indicating that the linear-response prediction continues to provide a good approximation even when as many as \(N/2\) nodes are perturbed or when the number of perturbed edges reaches approximately \(20\%\) of the total number of edges.

\subsection{Stress test for removal of nodes and edges}
\quad We treat removal events as large-perturbation stress tests of the linear-response framework rather than as validations of linear response theory itself. Unlike the multiple-perturbation tests above, these removal protocols are not infinitesimal, and therefore they are not expected to provide a quantitatively exact validation of the first-order theory. Instead, they are used to identify the regime in which linear response remains informative and the regime in which higher-order effects become important. Edge removal is implemented by setting \(W_{\alpha\beta}\to 0\), corresponding to \(\delta W_{\alpha\beta}=-W_{\alpha\beta}\). Node removal is implemented by isolating the selected node through the removal of all incoming and outgoing edges connected to it. Fig. \ref{MultipledKdW_remove} examines the removal of 40 directed edges in a directed Erd\H{o}s-R\'enyi network with nonuniform noise. The prediction is obtained by summing the single-edge response functions over all removed edges. The approximate agreement with the diagonal shows that, for this level of edge removal, the first-order response still captures the dominant covariance change. Fig. \ref{MultipledKdW_noderemove} considers the removal of a single node in an undirected network with nonuniform noise. Because this operation simultaneously deletes all edges connected to the node, it represents a large perturbation. The visible deviations from the diagonal are therefore expected and indicate the onset of higher-order response terms beyond first order.

Fig. \ref{NRMSE_Error_Remove} provides a complementary error-based assessment for removal-type perturbations. In Fig. \ref{NRMSE_Error_Remove}a, we randomly remove \(1\text{--}5\) undirected edges from an undirected Erd\H{o}s-R\'enyi network with \(N=100\), \(p=0.1\), and nonuniform noise strengths, and plot the NRMSE between the first-order prediction and the simulation result as a function of the number of removed edges. In Fig. \ref{NRMSE_Error_Remove}b, we examine node removal in a controlled manner by scaling all edges connected to a selected node according to \(\delta W_{\alpha\beta}=-\lambda W_{\alpha\beta}\), where \(\lambda=1\) corresponds to complete removal. The results show that the first-order response remains informative for edge removal as long as the number of removed edges is only a small fraction of the total, approximately \(2\%\) in the present example. For node weakening or removal the error grows rapidly once \(\lambda>0.2\), indicating that the resulting perturbation has already left the linear-response regime. This behavior highlights the limitation of the first-order framework for large structural perturbations and motivates the development of a nonlinear response theory.
